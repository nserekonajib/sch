How SchoolPay Sync Works

1. The system will fetch payments from SchoolPay API using your configured credentials.

2. For each payment, it will look up the student using their payment code.

3. If the student is found, the payment is recorded in your local database.

4. Students not found will be listed and can be downloaded for reference.
